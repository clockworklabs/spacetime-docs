# Rust Module SDK Reference

SpacetimeDB allows using the Rust language to write server-side applications. These applications are called **modules** and have access to a built-in database.

Rust modules are written with the the Rust Module SDK. They are built using [cargo](https://doc.rust-lang.org/cargo/), like any other Rust application, and deployed using the [`spacetime` CLI tool](https://spacetimedb.com/install). Rust modules can import any Rust [crate](https://crates.io/) that supports being compiled to WebAssembly.

The Rust module SDK reference documentation is automatically generated from the module's source code and can be found [here](https://docs.rs/spacetimedb/latest/spacetimedb/).
